# Movie Database

This is a simple movie database that stores information about movies I have seen or want to see.

## Features

* Stores information about movies, including title, director, release year, genre, rating, and watched status.
* Allows users to view, add, edit, and delete movies.
* Provides a search function to find movies by title, director, or genre.

## How to use

1. Clone the repository to your local machine.
2. Create a database and run the `CREATE DATABASE movie_database;` and `USE movie_database;` commands.
3. Import the data from the `moviesDB.sql` file into the database.
4. Run the `main.py` file to start the application.
